0. Данные подгужаются в граф(update event):
    - по зуму(как на картах). Т.е. уменьшаешь масштаб - подгрузка(если нет данных)
    - по свайпу в любую сторону. Но грузятся все данные для конечных ключей, а не для ключей в направлении 
      свайпа(увы, но никак не смог это сделать.) 
1. STARTUP
let graph = new CoronaGraph('cy',myNode, settings);
где:
    cy - id элемента где будет граф. См. /test/index.html
    myMode - моя нода в формате {id, state, image}
    settings - см. ниже
  graph.init(data);
  где data - начальные данные( мои потомки )
  пример данных в test/app.js

2. EVENTS
    click -   клик по ноде event = {id:nodeId}
    update -  event = {ids:arrIds}нужна новая порция данных для родителей с id из arrIds
              данные в граф graph.add(data)
    busy -    граф рисует картинку(можно нарисовать спиннер, сейчас просто гасится div графа)
    ready -   можно убирать спиннер или что еще

3. SETTINGS
    умолчания в graph/default.js

    - myNode - стили для моей ноды
    - node   - стили для нод по умолчанию
    - edges - стили для лучей
    - keyNode - стили для ключей
    - statuses - стили для состояний, соответствуют state в данных. Добавляются к
                  стилю ноды
